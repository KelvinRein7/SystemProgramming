Description of the program
--------------------------
This prograa take user input data such as evidence ID, room name,
emf or thermal or sound data, the time that the evidence was found (timestamp)
and store them in the evidence book as a note book to collect all the data.

List of Files
-------------
1. defs.h
2. evidence.c
3. init.c
4. main.c
5. README.txt

Running the program:

	Step 1: gcc -Wall -std=c99 -o main main.c evidence.c init.c -g


	Step 2: ./main
